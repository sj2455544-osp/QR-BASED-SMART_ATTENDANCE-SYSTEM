Why Choose QR-Attend?

Built for Modern Classrooms

Dynamic 3-Min Tokens

Every QR code generated expires automatically after 3 minutes. Students cannot share screenshots of the code with friends at home.

50m GPS Geofencing

Using the Haversine formula, the system calculates the exact distance between the Teacher's screen and the Student's mobile device.

Live MySQL Sync

Attendance logs are pushed instantly to the database, allowing teachers to view present students in real-time on their dashboard.

Seamless 3-Step Process

1
Teacher Generates Session

The teacher opens the dashboard and creates a unique QR code. Their current latitude and longitude are logged into the database securely.

2
Student Scans Code

Students scan the QR code using any smartphone camera. Their browser instantly requests HTML5 high-accuracy GPS permissions.

3
Instant Validation

If the student is within 50 meters and the 3-minute timer hasn't expired, attendance is marked. Otherwise, they are instantly blocked.

📌 Overview

The QR-Based Smart Attendance System is an advanced digital solution that automates attendance using QR Code technology and AI-based face recognition.

It eliminates manual errors, improves accuracy, and provides a secure, fast, and scalable attendance system for institutions and organizations.

✨ Key Features

🚀 QR Code-based attendance
🔐 Secure Authentication System
🧠 Face Recognition (AI Integration)
📊 Real-time Attendance Tracking
👨‍💼 Admin Dashboard
📁 Attendance History & Reports
⚡ Fast and Responsive UI

⚙️ Installation Guide
1️⃣ Clone Repository

git clone https://github.com/your-username/qr-attendance-system.git

cd qr-attendance-system

2️⃣ Setup Virtual Environment

python -m venv venv
venv\Scripts\activate (Windows)
source venv/bin/activate (Linux/Mac)

3️⃣ Install Dependencies

pip install -r requirements.txt

4️⃣ Run Migrations

python manage.py makemigrations
python manage.py migrate

5️⃣ Start Server

python manage.py runserver

👉 Open: http://127.0.0.1:8000/

⚡ How It Works

1️⃣ User registers in system
2️⃣ Unique QR Code is generated
3️⃣ QR is scanned for attendance
4️⃣ Face Recognition verifies identity (optional)
5️⃣ Attendance stored automatically

✔ Unique QR per user
